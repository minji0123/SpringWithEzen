Spring FrameWork

스프링 특징
- 자바 애플리케이션 프레임워크
- 경량 프레임워크(Light Weight FrameWork)
- 경량 컨테이너(Light Weight Container)
- 자바 빈(class object)_스프링 생성, 초기화, 실행, 소멸을 담당
- 제어의 역전(IoC, Inversion of Control)
	- 프레임워크 객체를 관리
	- 애플리케이션 사이의 느슨 결합
- 의존성 주입(DI, Dependency Injection)
	- 프레임워크가 객체를 생성하고 값을 지정
- 관점지향(AOP, Aspect Oriented Programming)
	- 핵심 기능과 부가 기능을 분리
	
	
스프링 구조
- Spring core
	- IoC
- Spring AOP
	- 관점 지향 기능 제공
- Spring DAO
	- JDBC
- Spring Context
	- Bean 에 접근 방법 제공
- Spring ORM
	- 하이버네이트, 마이바티스 같은 영속성 관련 연동 기능 제공
- Spring Web
	- 웹 어플리케이션 기능 제공
- Spring MVC
	- MVC 구현 기능 제공

POJO
Plane Old Java Object(순수 자바 객체)
